Sobre mim

Sou um desenvolvedor front-end júnior procurando por uma oportunidade. Meu foco é escrever HTML acessível, usando práticas modernas de CSS e um JavaScript limpo. Quando estou escrevendo código JavaScript, na maioria das vezes estou usando React, mas posso me adaptar à qualquer ferramenta se necessário. Moro em Guarulhos, SP, mas também seria feliz trabalhando remotamente, inclusive tenho experiência em times remotos. Quando não estou codando, poderá me achar fora de casa. Eu adoro estar próximo a natureza seja para uma caminhada, corrida ou ciclismo. Eu amaria se você desse uma olhada no meu trabalho.
